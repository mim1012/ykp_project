3️⃣ 완전한 판매관리 (가장 중요!)

📋 개통표 접근 방법

  1. 1. 왼쪽 사이드바에서 📋 아이콘 클릭
  2. 2. "완전한 개통표 AgGrid" 페이지 열림
  3. 3. 40개 모든 필드가 포함된 개통표 확인

📅 개통표 작성 방법

  1. 1. 오늘 버튼 클릭 → 오늘 날짜로 새 개통표 생성
  2. 2. ➕ 새 개통 등록 클릭 → 새 행 추가
  3. 3. 각 필드 입력:
    • • 판매자: 직원명
    • • 대리점: 이앤티/앤투윈 등
    • • 통신사: SK/KT/LG
    • • 모델명: 갤럭시 S24 등
    • • 고객명/전화번호: 고객 정보
    • • 액면가: 요금제 금액
  4. 4. 🔄 전체 재계산 클릭 → 자동 마진/세금 계산
  5. 5. 💾 일괄 저장 클릭 → 데이터베이스 저장

✨ 자동 계산 기능

액면가만 입력하면 정산금, 세금, 마진이 자동으로 계산됩니다!
